### Usage


### Installation

Loginapp requires [Node.js](https://nodejs.org/) v4+ to run.
pull repository, cd to project directory 

```sh
$ npm install
```
To start the app
```sh
$ node app.js
```
view resulting webpage in localhost:port (port is usually 3000 or 8080)
